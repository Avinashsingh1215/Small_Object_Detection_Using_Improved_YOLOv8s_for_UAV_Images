# Fully Integrated UAV Small Object Detection

## Features
- ✅ Lightweight PMSE (depthwise separable)
- ✅ SCFPN with P2 ultra‑small detection head
- ✅ Coordinated Attention after SCFPN
- ✅ WIoU loss (WIOUv3)
- ✅ Dynamic Label Assignment (fully integrated)
- ✅ Exponential Moving Average (EMA) callback
- ✅ Multi‑scale training (configurable)
- ✅ All paper hyperparameters (lr=0.0005, momentum=0.9, etc.)
- ✅ Checkpoint resume (auto‑detects `last.pt`)
- ✅ Live epoch metrics (loss, mAP50, precision, recall)
- ✅ Comprehensive plots (training curves, confusion matrix, PR, F1)
